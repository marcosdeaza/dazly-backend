┌─────────────────────────────────────────────────────────────────────┐
│  🆘 SOLUCIÓN DE PROBLEMAS                                            │
└─────────────────────────────────────────────────────────────────────┘

  Problema: "Sigo recibiendo 429"
  Solución:
    1. VERIFY_SETUP.bat
    2. Ver dashboard → ¿cuántas regiones activas?
    3. Si todas cooldown → esperar 1-2 min

  Problema: "Dashboard no carga"
  Solución:
    1. Verificar: dir public\stats-dashboard.html
    2. Reiniciar: npm start
    3. Verificar index.ts tiene express.static

  Problema: "Cache no funciona"
  Solución:
    • Solo cachea prompts IDÉNTICOS
    • Ver dashboard: "Entradas Caché" debe ser >0
    • Prueba mismo prompt 2 veces seguidas

  Problema: "Sistema lento"
  Solución:
    • Ver dashboard → cola muy larga?
    • Aumentar maxConcurrentRequests (3→5)
    • Ver tokens disponibles
